================================================================================
PROMPT ENGINEERING TECHNIQUES USED
================================================================================

1. Structured Output: JSON format for context selection ensures parseable responses
2. Role Definition: System roles establish expertise and context
3. Conditional Prompting: Different prompts based on query type and context availability
4. Content Truncation: 500-character previews optimize token usage
5. Format Specification: Markdown output format for readable responses
6. Temporal Injection: Current date/time added when needed
7. Fallback Instructions: Guidance when context is insufficient
8. Clear Boundaries: Explicit do/don't instructions prevent confusion
9. Positive Framing: Encourages helpful responses over refusals
10. Query Type Detection: Pattern matching identifies general knowledge questions
